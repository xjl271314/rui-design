# Notification组件设计思路

## 通知提醒框(Notification)组件一般会有如下需求点:

- 能控制Notification自动关闭的时间
- 能配置Notification渲染节点的输出位置
- 能控制Notification的弹出位置
- 能自定义关闭图标
- 可以手动选择通知窗类型
- 能自定义通知框的偏移量
- 能设置通知框的信息和提示文本
- 能自定义通知框的Icon
- 通知框点击时提供回调函数
- 通知框关闭时提供回调函数
- 能手动销毁通知框
